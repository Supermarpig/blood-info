## 三、根本原因分析

### 問題 1：域名沒有權威性

- ~~目前使用 `blood-info.vercel.app`（Vercel 子域名）~~ → **已換為 `www.bloodtw.com`（2026-03-27）**
- 沒有累積的反向連結（backlinks）
- 相比官方 `donate.blood.org.tw` 毫無競爭力

### 問題 2：首頁可見文字內容太少

- `app/page.tsx` 的 H1 只有「捐血活動查詢」5 個字
- 主要內容是動態載入的活動列表（JavaScript 渲染）
- Google 爬蟲看到的有效文字非常有限

### 問題 3：「今天哪裡有捐血車」標題沒對應到

- 這個查詢有 21 次曝光但 0 點擊
- 首頁 title/description 沒有直接包含這類問句型關鍵字
- 用戶看到搜尋結果沒有被吸引點擊

### 問題 4：地區頁缺少固定文字介紹

- `/region/[slug]` 頁面只有活動列表
- 缺少每個縣市的固定介紹段落（Google 喜歡有實質內容的頁面）

### 問題 5：長尾關鍵字頁面不足

- 沒有針對「今天哪裡有捐血車」這類問句型查詢的專屬頁面
- 缺少年份頁面（如「2026 捐血贈品」）

## 十二、整體 SEO 策略與流量天花板（2026-05-09 確認）

### 為什麼不搶「捐血」這個大詞？

「捐血」月搜尋量大，但競爭者是 `donate.blood.org.tw`（台灣血液基金會官方）、各縣市衛生局、維基百科——這些網站有數十年 domain authority 與大量外部連結，短期內無法超越。即使搶到第一頁，來自「捐血」的流量品質也低（使用者只是在找資訊，不一定使用查詢工具）。

### 現行策略：長尾意圖詞 + topic cluster

鎖定使用者有明確行動意圖的詞：

| 詞型 | 範例 | 優勢 |
| ---- | ---- | ---- |
| 活動查詢型 | 捐血活動查詢、近期捐血活動 | 使用者會直接用查詢功能 |
| 地點型 | 今天哪裡有捐血車、今天哪裡有捐血活動 | 即時需求，高轉換率 |
| 城市 × 贈品型 | 新北捐血活動贈品、台中捐血贈品查詢 | 競爭少、CTR 高達 9–11% |
| 年份標籤型 | 捐血贈品查詢2026 | CTR 高達 26%，可複製到其他詞 |

**執行邏輯：** 每個城市贈品文章、每個專頁都是獨立 URL，可單獨排名。多個長尾詞加總，加上 topic cluster 內部連結讓 Google 理解主題權威，最終帶動整體排名前移。

### 流量天花板與成長路徑

| 階段 | 月點擊估算 | 需要做什麼 |
| ---- | ---------- | ---------- |
| 現在（2026-05） | ~700–800 | 已達成 |
| 短期目標 | 3,000–5,000 | 城市贈品 topic cluster 補完、排名從 7.6 推進前 5 |
| 中期目標 | 5,000–10,000 | 外部連結（PTT、Dcard、媒體報導）開始累積 |
| 長期目標 | 10,000+ | domain authority 足夠後，中等競爭詞（捐血活動、近期捐血活動）能穩居前 3 |

光靠站內 SEO 上限約 3,000–8,000/月；突破一萬需要外部連結配合。
