## 三、根本原因分析

### 問題 1：域名沒有權威性

- 目前使用 `blood-info.vercel.app`（Vercel 子域名）
- 沒有累積的反向連結（backlinks）
- 相比官方 `donate.blood.org.tw` 毫無競爭力

### 問題 2：首頁可見文字內容太少

- `app/page.tsx` 的 H1 只有「捐血活動查詢」5 個字
- 主要內容是動態載入的活動列表（JavaScript 渲染）
- Google 爬蟲看到的有效文字非常有限

### 問題 3：「今天哪裡有捐血車」標題沒對應到

- 這個查詢有 21 次曝光但 0 點擊
- 首頁 title/description 沒有直接包含這類問句型關鍵字
- 用戶看到搜尋結果沒有被吸引點擊

### 問題 4：地區頁缺少固定文字介紹

- `/region/[slug]` 頁面只有活動列表
- 缺少每個縣市的固定介紹段落（Google 喜歡有實質內容的頁面）

### 問題 5：長尾關鍵字頁面不足

- 沒有針對「今天哪裡有捐血車」這類問句型查詢的專屬頁面
- 缺少年份頁面（如「2026 捐血贈品」）

## 四、改善行動計畫

### 🔴 高優先（直接影響排名）

#### 1. 換自訂域名

```
blood-info.vercel.app → 買一個 .tw 或 .com 域名
例如：bloodinfo.tw / 捐血活動.tw
```

Vercel 子域名的 SEO 本質上是從零開始，換域名才能累積權威。

#### ✅ 2. 首頁加入關鍵字豐富的介紹段落

已在 `app/page.tsx` H1 下方加入靜態介紹文字，涵蓋台北、新北、台中、高雄等縣市關鍵字。

#### ✅ 3. 修正「今天哪裡有捐血車」的 meta description

已更新 `app/layout.tsx` 的 description、OpenGraph、Twitter Card，全部改為「今天哪裡有捐血車？」開頭的問句型描述。

### 🟡 中優先（強化現有頁面）

### 🟢 低優先（長期積累）

#### 7. 增加外部反向連結

- 在 PTT (血液板、省錢板) 分享網站
- 在 Dcard 健康/公益版分享
- 聯絡捐血相關社群/粉絲頁介紹

#### ✅ 8. ISR 取代 `cache: "no-store"`

已更新 `app/page.tsx`、`app/region/[slug]/page.tsx`、`app/gift/[slug]/page.tsx`：

```typescript
// 改前
cache: "no-store";

// 改後（每小時快取一次）
next: {
  revalidate: 3600;
}
```

Google 爬蟲每次抓取都能看到完整的 HTML 內容。

## 六、競爭分析

你的主要競爭對手：

| 網站                                  | 優勢                   |
| ------------------------------------- | ---------------------- |
| donate.blood.org.tw（台灣血液基金會） | 官方網站，極高域名權威 |
| 各縣市衛生局網站                      | 地方政府，本地 SEO 強  |

**策略建議：不要跟官方硬碰主關鍵字，改走「長尾+地區+贈品」的利基市場。**

例如：

- ✅ `台北捐血贈品查詢 2026`（競爭少）
- ✅ `今天新北哪裡有捐血車`（問句型，競爭少）
- ❌ `捐血`（競爭太強，不切實際）
